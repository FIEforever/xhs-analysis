如何启动 xiaohongshu-mcp 服务

    

      
本系统通过 MCP 协议调用小红书官方数据接口。请按以下步骤启动服务：

      
方式一：使用 Docker（推荐）

      
docker pull xpzouying/xiaohongshu-mcp
# 下载 docker-compose 文件
curl -O https://raw.githubusercontent.com/xpzouying/xiaohongshu-mcp/main/docker/docker-compose.yml
# 启动服务
docker compose up -d

      
方式二：直接运行二进制文件

      
# 1. 先运行登录工具
./xiaohongshu-login
# 2. 启动 MCP 服务
./xiaohongshu-mcp

      
服务启动后默认监听 http://localhost:18060/mcp，本系统将自动连接。

      
⚠️ 如服务未启动，系统将使用内置的演示数据展示分析功能。
